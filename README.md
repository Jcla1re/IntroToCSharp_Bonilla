# IntroToCSharp_Bonilla
Prelim Lab 01 - Codac Logistics Fuel Tracking Tool

Description:
This console-based C# application simulates a fuel and performance tracking system for a logistics company. 
The program collects driver information, validates weekly travel distance, records five days of fuel expenses, 
analyzes fuel efficiency, and generates a structured weekly audit report.

The system ensures accurate financial tracking and evaluates whether the driver stayed within the allocated fuel budget.

Features:
- Collects driver profile information
- Validates weekly distance input
- Records 5-day fuel expenses
- Calculates total and average fuel cost
- Evaluates fuel efficiency rating
- Determines budget compliance
- Generates a formatted weekly audit report

Concepts Applied:
- Console input and output
- Data validation
- Arrays
- Looping structures
- Conditional statements
- Basic financial calculations
- Structured program organization
